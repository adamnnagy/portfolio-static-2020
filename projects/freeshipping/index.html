Free Shipping VR | 2019
Previous Next

Free Shipping was my thesis project at NYU Abu Dhabi. The player makes decisions in a Virtual Reality environment while having to use unconscious biases.
"In this Virtual Reality experience, the player embodies the job of a machine learning algorithm supervisor. They will not only figure out which object goes to which ethnicity but also how you interact with their unconscious biases."
I developed the game using open source tools, including Unreal Engine 4.20 and its Blueprint Programming environment, Blender, Google Blocks.
